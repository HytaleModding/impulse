# Support

Impulse is alpha software. APIs and behavior may change while Hytale's pre-release server API
stabilizes.

Use GitHub issues for reproducible bugs, documentation gaps, and focused feature requests.

Include:

- Impulse commit or version;
- Hytale server version;
- backend id, such as `impulse:bullet` or `impulse:rapier`;
- operating system and architecture when native artifacts are involved;
- reproduction steps and validation commands already run.

Questions about large architecture changes should start as a design discussion before a pull
request.
